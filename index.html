Configurar Chave Groq API

Insira sua chave de API do Groq para conversar com o WSM 1.0 em tempo real. A chave é salva apenas localmente no seu navegador.

💡 Dica para Vercel: Configure a variável de ambiente GROQ_API_KEY nas Settings do seu projeto. Pode ser feito manualmente aqui ou automaticamente se houver um endpoint /api/config que retorna {groqApiKey: "..."}.
